# Employee Management System

## Overview

Employee Management System is a web application built using ASP.NET Core MVC, SQL Server, Dapper, and Stored Procedures for managing employees and departments.

## Features

* Authentication and Authorization
* Role-Based Access Control (Admin/User)
* Employee Management (CRUD)
* Department Management (CRUD)
* Dashboard Statistics
* Employee Search
* Pagination
* Employee Photo Upload
* Image File Validation
* SQL Server Database
* Dapper ORM
* Stored Procedures

## Technologies Used

* ASP.NET Core MVC
* C#
* SQL Server
* Dapper
* Bootstrap 5
* HTML, CSS, JavaScript

## Project Structure
